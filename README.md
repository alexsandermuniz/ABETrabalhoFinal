# ABETrabalhoFinal

Instruções para rodar no DOCKER:

Clone o repositório, acesse a pasta ABETrabalhoFinal.

Execute docker-compose up

Dois serviços irão subir, um na porta 4012 e outro na porta 4013:
http://localhost:4012/wholesaleapi/v1/
http://localhost:4013/shopkeeperapi/v1/
